# Contributing

See [docs/CONTRIBUTING.md](docs/CONTRIBUTING.md) for the full guide.

Job-Hunt uses Codex plus the checked-in repo instructions as the primary
contributor workflow. If you are still getting your local environment ready,
start with [docs/SETUP.md](docs/SETUP.md). If you are blocked on setup, bug
reporting, or security routing, use [docs/SUPPORT.md](docs/SUPPORT.md).

Quick rules:

- keep changes small and factual
- do not commit user-layer data
- update docs when behavior changes
- run the relevant validation scripts before review
