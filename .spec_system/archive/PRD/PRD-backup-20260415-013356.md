# jobhunt - Product Requirements Document

## Overview

AI-powered job search pipeline

## Goals

1. Help candidates evaluate job opportunities with structured, high-signal analysis.
2. Generate tailored, ATS-oriented application materials from user profile data.
3. Maintain a reliable application pipeline with scanning, tracking, and integrity checks.
